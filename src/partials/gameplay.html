Core Mechanics

A system that moves even when you don’t.
Shapes drift slowly toward the unstable center
You can only place objects within the outer margin
Every level provides a limited number of placements
Everything you do changes how the whole thing behaves
You don’t push objects directly. You influence trajectories from the border. Each placement changes how the system breathes, shifts, and settles.
Sometimes the smartest thing you can do is just... nothing.
